실습3_배열

문제 1

아래의 조건에 맞는 sum 함수를 작성한다.
prompt로 사용자에게 숫자 입력을 반복 요청한 후, 입력받은 값들을 배열에 저장한다.
숫자가 아닌 값(isNaN(value) 함수는 value가 숫자가 아니면 true, 숫자이면 false 리턴) 혹은 빈 문자열을 입력하거나 취소 버튼을 누르면 prompt 창의 반복을 멈추고 배열에 담긴 숫자의 합계를 반환한다.
(단, 숫자 0은 유효한 입력으로 간주한다.)

문제 2

값 추가 버튼을 눌렀을 때 뜨는 prompt에 입력 된 값을 배열에 순서대로 저장한다.
단, 취소 및 값 입력 없이 확인을 눌렀을 경우는 저장하지 않는다.
값 출력 버튼을 누르면 지금까지 입력 된 값을 alert로 출력한다.
단, 아직 입력 된 값이 없는 상태에서 버튼이 눌리면 '값이 먼저 입력 되어야 합니다.'를 출력한다.
값 삭제 버튼을 누르면 지금까지 입력 되었던 배열의 값을 모두 제거한 뒤 '값이 모두 삭제되었습니다.'를 출력한다.

문제 3

배열 arr의 요소 중 a와 b 사이에 속하지 않는 요소는 삭제해주는 함수 filterRange(arr, a, b)를 작성한다.
작성한 함수는 기존 배열을 변경하기만 하고 아무것도 반환하지 않는다.

문제 4

아래 선언 된 변수를 매개변수로 전달 받았을 때 alert 창으로 "loveyou"가 출력 되게 하는 reverseWords 함수를 정의하여 호출 테스트한다.
별도의 문자열을 사용하면 안되며 배열에서 학습한 메소드를 활용하여 작성한다.

문제 5

아래 선언 된 변수를 "," 기준으로 분리하여 선언 된 순서대로 하나의 배열로 결합한다.
prompt로 사용자에게 입력 받은 값이 해당 배열에 존재하면 alert를 통하여 배열 내 위치를 출력하고 존재하지 않으면 alert로 존재하지 않습니다를 출력하는 findWord 함수를 완성한다.
단, prompt 창에서 취소 또는 입력 값 없이 확인을 눌렀을 경우는 결과를 확인할 수 없습니다를 alert를 통하여 출력한다.

Ex. 사용자 prompt 입력 값 : 사과, alert 결과 값 : 사과는 배열 index 0번째에 있습니다.
Ex. 사용자 prompt 입력 값 : 강아지, alert 결과 값 : 강아지는 배열 index 11번째에 있습니다.
Ex. 사용자 prompt 입력 값 : 햄스터, alert 결과 값 : 햄스터는 배열에 존재하지 않습니다.
Ex. 사용자 prompt 입력 시 취소 또는 빈칸 입력, alert 결과 값 : 결과를 확인할 수 없습니다.
